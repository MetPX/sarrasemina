Notes de publication pour sarracenia/dd_subscribe

-------------------------------------------------------------------------------

Sarra Semina est le compagnon Web de Sarracenia.

Il permet aux usagers de construire des configurations personnalisées en 
prévisualisant les produits que Sarracenia fournirait avec celle-ci.

Sarra Semina est alimenté quotidiennement avec les données de Sarracenia.
Deux scripts Python construisent les catalogues de produits selon la source
utilisée :
- sarra_catalogr.py explore l'arborescence d'un répertoire
- sarra_rebuildr.py lit un fichier tar tvz obtenu de la sauvegarde sur ruban

-------------------------------------------------------------------------------
- Utilisation
-------------------------------------------------------------------------------

sarra_catalogr.py source_path_folder output_path_folder
sarra_rebuildr.py source_path_file   output_path_folder
